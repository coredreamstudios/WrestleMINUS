WrestleMINUS

A file reader and editor for Yukes wrestling games

Current Version: 1.3.3.0 (Download)

About WrestleMINUS

This is a program merging several smaller tools I've previously made into an all in one file reader and editor.

The tool aims to be primarily used by users already familar with editing Yukes games, however there is the hope that a streamlined process will lower the barrier to entry.

Prerequisites

Asmodean's unrrbpe is a standalone exe that decompresses bpe. (Download Here.) This may be later removed as a prerequisite

texconv.exe is a dds converter exe used for that purpose with this tool. This exe will be released with each major update the complete source can be found ( Here.)

Ionic.Zlib.dll is a Zlib compressor and decompressor used for that purpose with this tool. This dll will be released with each major update the complete source can be found ( Here.)

Bink's RAD video tools. (Download Here.) This is used for opening Bink video files ".bk2" WrestleMINUS will try to autodetect this exe.

DDS Editing tools can also be selected to quickly open and edit textures. Options include: (Paint.net) (Gimp) (Adobe Photoshop.)
